★ Datos Agrupados - Ingreso (USD)

📋 Contexto: Trabajo Social Ecuador

Ingreso mensual familiar (USD) de 80 familias.

Datos originales:
Valores entre 150 y 550 USD

PASO 1: RANGO

Rango = Máx - Mín = 550 - 150 = 400
¿Por qué? Mide la dispersión total de los datos.

PASO 2: Intervalos (K)

K = 1 + 3.322 log(80) = 7.32 ≈ 8
Fórmula de Sturges: Determina número óptimo de grupos para n datos.

PASO 3: Amplitud (C)

C = Rango / K = 400 / 8 = 50
¿Por qué 50? Tamaño de cada intervalo.
